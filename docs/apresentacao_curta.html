Retail Analytics — Apresentação Curta

Projeto de estudo end-to-end para demonstrar competências em SQL, modelação relacional, transformação com dbt e lógica de base de dados no estilo Oracle PL/SQL.

1) Problema e objetivo

  • Simular uma operação de retalho com dados transacionais realistas.
  • Transformar dados operacionais em insights analíticos.
  • Estruturar a solução de forma explicável para contexto profissional.

2) Arquitetura resumida

  • SQLite: camada operacional (OLTP simplificada).
  • DuckDB: camada analítica local (OLAP).
  • dbt: transformação, testes e documentação.
  • PL/SQL concepts: treino de lógica Oracle-style.

3) Entregáveis

  • Guia para iniciantes e roadmap de estudo.
  • Guia de instalação completo com troubleshooting.
  • Documento de análise profunda (arquitetura, riscos, evolução).
  • Requisitos técnicos em requirements.txt e metadados em package.json.
