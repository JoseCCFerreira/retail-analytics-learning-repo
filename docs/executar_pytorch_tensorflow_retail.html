Como Instalar e Executar PyTorch e TensorFlow no Retail Case

Guia prático com instalação, apoio teórico e casos reais de execução sobre o dataset do projeto.

Instalar Teoria PyTorch TensorFlow Casos Praticos Executar Outputs

1. Instalação

Primeiro prepara a base do projeto:

python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python python/run_pipeline.py

Depois instala a stack opcional de Deep Learning:

pip install -r requirements_deep_learning.txt
PyTorch e TensorFlow podem demorar mais a instalar e ocupam mais espaço. Faz sentido mantê-los numa camada opcional, separada do projeto base.

2. Apoio Teórico

Regressão Aqui queres prever um valor numérico. No projeto, o alvo principal é net_amount.
Classificação Aqui queres prever uma classe. No projeto, o alvo é o customer_segment.
Rede neuronal densa Boa escolha para começar quando tens dados tabulares com features numéricas e categóricas codificadas.
Baseline primeiro A metodologia certa continua a ser comparar sempre com scikit-learn antes de assumir que DL é melhor.

3. Caso prático com PyTorch

Ficheiro: deep_learning/pytorch_retail_examples.py

Este script faz duas coisas:

  • treina uma rede neuronal simples para regressão do net_amount;
  • treina uma rede neuronal simples para classificação do segmento do cliente.

Executar

python deep_learning/pytorch_retail_examples.py

O que aprendes aqui

  • como transformar arrays em tensores;
  • como criar DataLoader;
  • como definir uma MLP;
  • como usar loss, optimizer e epochs;
  • como guardar métricas e pesos do modelo.

4. Caso prático com TensorFlow / Keras

Ficheiro: deep_learning/tensorflow_retail_examples.py

Este script faz o mesmo tipo de problema, mas usando Keras.

Executar

python deep_learning/tensorflow_retail_examples.py

O que aprendes aqui

  • como declarar um modelo em Sequential API;
  • como compilar com optimizer, loss e metrics;
  • como treinar com fit();
  • como prever com predict();
  • como guardar um modelo .keras.

5. Casos práticos para estudar a seguir

Caso Objetivo Primeira metodologia
Previsão de valor da compra Estimar net_amount de uma linha de venda. Random Forest vs MLP.
Previsão do segmento Antecipar se o cliente é budget, family, premium ou business. Logistic Regression vs MLP.
Previsão de receita mensal Entrar no mundo temporal. Baseline de média móvel, depois LSTM/TCN.
Recomendação de produto Produtos mais prováveis para cada cliente. Top-N simples, depois embeddings.

6. Ordem certa para executar

  1. python python/run_pipeline.py
  2. python python/ml_retail.py
  3. pip install -r requirements_deep_learning.txt
  4. python deep_learning/pytorch_retail_examples.py
  5. python deep_learning/tensorflow_retail_examples.py
