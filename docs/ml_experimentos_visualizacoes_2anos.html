Retail: ML, Comparacao, Visualizacoes e Dados de 2 Anos

Guia para expandir o caso Retail com mais experiencias de Machine Learning, validacao comparavel e novas visualizacoes no dashboard.

Hub · Codigo + outputs · Uniformizacao

1. Gerar dados dos ultimos 2 anos

No Retail os dados sao sinteticos, portanto "descarregar" equivale a gerar uma janela temporal de 2 anos terminando na data atual da maquina.

python python/run_pipeline.py --years 2 --transactions 25000
python python/ml_retail.py
streamlit run streamlit/app.py
Para validacao rapida usa 5.000 a 25.000 transacoes. Para stress test podes subir para 100.000, mas o ML e o dashboard ficam mais pesados.

2. Casos de ML que podes testar e comparar

CasoObjetivoModelos candidatosMetricasVisualizacao
Prever valor liquido da venda Estimar net_amount por linha de venda. Linear Regression, Random Forest, Gradient Boosting, XGBoost/LightGBM se adicionares dependencias. MAE, RMSE, R2, cross-validation. Feature importance, erro real vs previsto, residuos por categoria.
Classificar segmento de cliente Prever customer_segment com comportamento de compra. Logistic Regression, Random Forest, Gradient Boosting, SVM. Accuracy, macro F1, confusion matrix. Matriz de confusao e barras por segmento.
Segmentar clientes Descobrir grupos sem usar label. K-Means, DBSCAN, Gaussian Mixture, Agglomerative Clustering. Silhouette, Davies-Bouldin, estabilidade por seed. PCA/UMAP scatter com cor por cluster.
Prever probabilidade de desconto Estimar se uma linha tem desconto. Logistic Regression, Random Forest, Gradient Boosting. ROC AUC, Precision, Recall, F1. Curva ROC, precision-recall e importancia das features.
Forecast de receita mensal Projetar receita futura por mes. Moving average, Exponential Smoothing, Prophet/ARIMA se adicionares dependencias. MAE temporal, MAPE, backtesting. Linha historico vs forecast com intervalo.
Anomalias comerciais Detetar vendas fora do padrao. Isolation Forest, Local Outlier Factor, z-score por categoria. Taxa de outliers, inspecao manual, estabilidade. Scatter quantidade vs valor com outliers destacados.

4. Como validar e comparar

python python/run_pipeline.py --years 2 --transactions 25000
python python/ml_retail.py
python python/validate_project.py
streamlit run streamlit/app.py

Guarda os outputs de data/ml_outputs/ por experiencia. Compara sempre a mesma janela temporal, mesmas features e mesma seed antes de decidir que um modelo e melhor.
